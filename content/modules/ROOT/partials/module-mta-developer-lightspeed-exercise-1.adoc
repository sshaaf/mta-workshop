[IMPORTANT]
====
In this lab, we will use the **Migration Toolkit for Applications** Visual Studio Code extension—the **MTA extension** in a Red Hat DevSpaces environment that provides a VSCode-based GUI.

**Developer Lightspeed** is not a separate extension. It is MTA's AI-powered code refactoring feature. The **MTA extension** is what is installed in the IDE; it includes both MTA static code analysis and Developer Lightspeed (generative AI).
====

.Goal
[NOTE]
====
Prepare the **OpenShift Dev Spaces** workspace and **MTA extension** for the rest of the lab: open the **Developer Lightspeed-coolstore** project, create a **migration profile** (source and target technologies), and connect **Developer Lightspeed** to the LLM via **GenAI** / **provider-settings.yaml** so you can analyze and modernize ACME’s applications with AI assistance.
